JS Hero Exercise #67: "while loop"

Link to JS Hero exercise page

Exercise Details:

Write a function spaces that takes a natural number n and returns a string of n spaces.

Example: spaces(1) should return ' '.

Test Function:

const spaces = n => {
  let result = '';
  while (result.length < n) {
    result += ' ';
  }
  return result;
};
