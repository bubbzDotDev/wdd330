Exercise Details:

Write a function digitsum that calculates the digit sum of an integer. The digit sum of an integer is the sum of all its digits.

Example: digitsum(192) should return 12.

Test Function:

const digitsum = (num) => {
  const numString = `${num}`;
  const numLength = numString.length;
  const numArray = [];
  let result = 0;

  for (let i = 0; i < numLength; i++) {
    numArray.push(numString[i]);
  }

  numArray.forEach(digit => {
    result += parseInt(digit);
  });

  return result;
}
