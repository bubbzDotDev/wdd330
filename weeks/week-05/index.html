JS Hero Exercise #64: "for loop"

Link to JS Hero exercise page

Exercise Details:

Write a function addTo that accepts a number as a parameter and adds all natural numbers smaller or equal than the parameter. The result is to be returned.

Example: addTo(3) should return 6 (1+2+3).

Test Function:

const addTo = (num) => {
  let result = 0;
  for (let i = 1; i <= num; i++) {
    result += i;
  }
  return result;
}
