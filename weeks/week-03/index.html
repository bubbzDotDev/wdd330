JS Hero Exercise #66: "Loops and arrays"

Link to JS Hero exercise page

Exercise Details:

Write a function mean that accepts an array filled with numbers and returns the arithmetic mean of those numbers.

Example: mean([1, 2, 3]) should return (1+2+3)/3 = 2.

Test Function:

const mean = (numArray) => {
  const length = numArray.length;
  let sum = 0;
  numArray.forEach(num => {
    sum += num;
  });
  return sum / length;
}
